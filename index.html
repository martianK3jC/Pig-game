Welcome to the Pig Game

This game requires 2 players

How to Play

In this game, two players take turns rolling a die. The goal is to be the first player to reach 50 points.
On each turn, a player can roll the die as many times as they want, but if they roll a 1, they lose all points for that turn.
If they roll any other number, they can choose to either keep the points or roll again.
The first player to reach 50 points wins!

About

Pig Game is a simple turn-based dice game built using HTML, CSS, and JavaScript. It's designed for learning and practicing core JavaScript concepts such as:

  • DOM Manipulation
  • Event Handling
  • State Management
  • Conditional Logic and Loops
  • Functions and Game Flow Control
